# Minimalist Focus Timer

Bu uygulama oemerfarukdev youtube kanalında canlı olarak Gemini Pro ile geliştirilmiştir.

## Teknik Detaylar

- Teknoloji: HTML, CSS, Javascript 
- Veri: localStorage
- Tasarım: minimalizm
- Platform: Responsive Web App 


### Prompt

```
Sen deneyimli bir frontend geliştiricisi ve UI/UX tasarımcısısın.

Görev: Benim için "Clear Focus" adında, tek bir HTML dosyasında çalışacak (HTML+CSS+JS iç içe), modern ve minimalist bir odaklanma (Promodo) uygulaması kodla.

Teknik Gereksinimler:
 1. Yapı: Promodo sayacı (25 dk) + Yapılacaklar Listesi (To-Do list)
 2. Veri: Tüm veriler (görevler ve sayaç durumu) tarayıcının localStorage kısmında saklanmalı. Sayfa yenilense bile veri kaybolmamalı.
 3. Responsive: Hem masaüstü hem de mobil cihazlarda kusursuz görünmeli (mobile-first yaklaşımı)

Tasarım Dili (önemli):
 - Stil: ultra minimalist. Karmaşık renkler yok. Siyah, beyaz ve vurgu rengi olarak yumuşak bir "pastel mavi" veya "turuncu" kullan
 - Tipografi: Sans-serif, temiz ve okunaklı fontlar (Inter veya Roboto)
 - Animasyonlar: Butonlara tıklandığında, görev eklendiğinde veya sayaç bittiğinde çok. yumuşak (smooth) css geçişleri ve mikro animasyonlar olsun. Kaba hareketler istemiyorum.
 - Görünüm: Kart yapısı, hafif gölgeler (soft shadows) ve yuvarlatılmış köşeler (border-radius) 

lütfen tüm kodu tek bir blokta ver
```
